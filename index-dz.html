Куратори проекту?

Master of Code

Master Of Code Global – це велика міжнародна IT-компанія, в якій працює близько 150 фахівців. Була заснована в 2004 році. Компанія є відомою у сфері розробки чат-ботів, а також спеціалізується на розробці веб та мобільних додатків. Нещодавні проекти: інформаційний чат-бот ONA17, інтернет магазин eBags, додаток для поширення фотографій Рhox та інші.

SPD

SPD-Ukraine (Software Product Development Ukraine) – найбільша IT-компанія в Черкасах, в якій працює близько 250 спеціалістів. Була заснована в 2006 році. Спеціалізується на Java проектах. Фірма має ряд дуже великих проектів, серед яких можна назвати, наприклад, PitchBook.

Interlink

Interlink – провідна черкаська IT-компанія, в якій працює близько 100 розробників. Була заснована в 2000 році. Компанія працює з великим набором технологій: Java, JavaScript, Node.js, Ruby on Rails, Python, Go, C# и Objective C та інші. Найвідоміші проекти: Cargo Bridge (250к завантажень з Apple Store), туристичний сервіс SPO, соціальна мережа Strbrd та інші.

Статистика розробки

  • було створено 2919 комітів
  • реалізовано 119 issue на front-end’і та 136 issue на back-end’і
  • створено близько 360 класів та інших артефактів на back-end’і та приблизно 460 на front-end’і
  • близько 12-ти студентів, що брали участь в проекті, зараз працюють в IT-фірмах.

Робота в команді та SCRUM

Angular

Angular - фреймворк з відкритим вихідним кодом, написаний на TypeScript і підтримуваний компанією Google. Він являє собою повністю переписану версію свого популярного попередника, AngularJS. За допомогою Angular можна розробляти програми на JavaScript (застосовуючи синтаксис ECMAScript 5 або 6), Dart або TypeScript.

TypeScript

TypeScript є надбезліччю JavaScript, універсальна мова, створеним в кінці 2012 року співробітником Microsoft Андерсом Хейслбергом - автором C #.

Основні переваги TypeScript
  1. Компілюється в нормальний JS
  2. Підтримка класів, інтерфейсів і нормального ООП
  3. Сувора типізація, IDE буде давати підказки та підсвічувати помилки, якщо ви, наприклад, передали в якості аргументу рядок замість числа
  4. Відсутність CallBack Hell і сотні})})})})})} завдяки Async / Await;
  5. Наявність всіх нових можливостей в JS, про які ви чули, але поки не можете використовувати через слабку підтримки браузерами

Boostrap

Bootstrap - бібліотека компонентів для створення інтерфейсу користувача з відкритим вихідним кодом, розроблена компанією Twitter. Вони будуються відповідно до принципів адаптивного веб-дизайну, що значно підвищує цінність бібліотеки, якщо ваш веб-додаток повинен автоматично подстроювати свій макет в залежності від розміру екрану пристрою користувача.

RxJS

Набір бібліотек, необхідних для створення асинхронних програм і програм, заснованих на подіях, з використанням спостережуваних колекцій. Дозволяє програмам працювати з асинхронними потоками даних на зразок серверного потоку котирувань акцій або подій, пов'язаних з рухом миші. За допомогою RxJS потоки даних представляються у вигляді спостережуваних послідовностей. Цю бібліотеку можна застосовувати як з іншими фреймворками JavaScript, так і без них.
